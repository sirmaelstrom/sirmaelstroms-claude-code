`clamp` constrains a number to a range. Given a `value`, a lower bound `min`, and an upper bound `max`, it returns `value` if it's already within `[min, max]`, `min` if `value` is below the range, or `max` if `value` is above it.

The inner `Math.min(max, value)` caps the value at `max`, and the outer `Math.max(min, ...)` floors it at `min`. The two calls compose to enforce both bounds in a single expression.
